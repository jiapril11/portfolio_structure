package.json

번들링에 필요한 npm 모듈들을 기술

src /

개별적인 컴포넌트 작업폴더 모음

gulpfile.js

개별적인 컴포넌트 작업폴더들을 프로젝트로 폴더를 빌드하기 위한 gulp 설정 파일

node_modules /

package.json을 통해서 설치가 되는 모듈 모음 폴더

dist /

gulp.js의 설정을 통해서 src폴더의 컴포넌트 파일들이 배포버전의 파일들로 빌드되는 완성 폴더

js / combined.js
각 섹션별, 페이지별 모든 js파일들을 하나의 파일로 합친버전

기존 combined.js를 다시 압축한 버전 (최종 배포파일)
static /
